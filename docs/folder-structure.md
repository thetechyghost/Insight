## `/convex/` — Backend (Convex BaaS)

All operational data. Real-time subscriptions. Multi-tenant, RBAC-enforced.

```
convex/
├── schema.ts                          # DATABASE SCHEMA — source of truth for all tables
├── crons.ts                           # Scheduled/recurring jobs
├── http.ts                            # HTTP API endpoints
├── testing.ts                         # Test utilities exposed to convex-test
│
├── lib/                               # Core infrastructure (DO NOT bypass)
│   ├── auth.ts                        # Identity → User resolution
│   ├── customFunctions.ts             # tenantQuery, tenantMutation, authedQuery, authedMutation
│   ├── platformFunctions.ts           # platformAdminQuery, platformAdminMutation wrappers
│   ├── rbac.ts                        # Permission constants + enforceRole()
│   ├── tenancy.ts                     # Membership verification + role hierarchy
│   ├── validators.ts                  # Shared Convex arg/return validators
│   └── helpers.ts                     # Misc utility helpers
│
├── test/
│   └── setup.ts                       # seedUserWithTenant, test identity helpers
│
└── _generated/                        # Auto-generated by Convex CLI — never edit
```
